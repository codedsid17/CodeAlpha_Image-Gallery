About This Gallery

Built with care, by a curious developer.

This gallery was designed and developed by Siddhi as a front-end project. The goal was to combine modern web design with interactive user experiences through responsive layouts, smooth animations, image filtering, and lightbox functionality.

Every section of the website focuses on usability, aesthetics, and performance. The project demonstrates skills in HTML, CSS, and JavaScript while creating an engaging platform for showcasing visual content.

◈

Responsive Design

Layouts adapt seamlessly across desktop, tablet, and mobile screens using fluid grids and flexible components.

Modern UI/UX
