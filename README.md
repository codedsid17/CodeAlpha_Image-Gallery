# ✨ Lumière

Lumière is a modern and visually appealing image gallery built with HTML, CSS, and JavaScript. Designed with a focus on aesthetics and user experience, it provides a clean and responsive space to showcase beautiful photography and visual content.

## 🎨 Inspiration
The name *Lumière* means **"light"** in French, reflecting the essence of photography and visual storytelling. This project was created to explore frontend design, responsive layouts, and creating immersive user experiences.

## 🌟 Features
- Responsive gallery layout
- Clean and elegant design
- Smooth browsing experience
- Optimized for desktop and mobile
- Lightweight and fast
